## ============= Interview Questions =============

### HTML

1.HTML কী এবং এটি কীভাবে কাজ করে?

- ব্যাখ্যা: HTML (HyperText Markup Language) হলো ওয়েব পেজ তৈরির জন্য ব্যবহৃত একটি মার্কআপ ভাষা। এটি বিভিন্ন ট্যাগ (tags) ব্যবহার করে ওয়েবসাইটের কনটেন্ট গঠন করে। HTML ব্রাউজারের জন্য নির্দেশনা দেয় কিভাবে টেক্সট, ইমেজ, ভিডিও ইত্যাদি প্রদর্শন করতে হবে।

2. ওয়েবপেজের স্ট্রাকচার কী?

- ওয়েবপেজের স্ট্রাকচার (Structure) মূলত HTML, CSS, এবং JavaScript এর সমন্বয়ে গঠিত হয়।

- ওয়েবপেজের স্ট্রাকচার ব্যাখ্যা:
  1️⃣ HTML মূলত ওয়েবপেজের মূল স্ট্রাকচার তৈরি করে। এটি বিভিন্ন ট্যাগ ব্যবহার করে কনটেন্ট সাজায়।
  2️⃣ CSS ওয়েবপেজকে সুন্দর ও আকর্ষণীয় করে তোলে।
  3️⃣ JavaScript ওয়েবপেজে ডায়নামিক ফিচার যোগ করতে ব্যবহৃত হয়।

- একটি সাধারণ ওয়েবপেজের মূল স্ট্রাকচার নিচের মতো হয়ঃ
  1️⃣ <!DOCTYPE html> → HTML ডকুমেন্টের ধরন নির্ধারণ করে।
  2️⃣ <html> → সম্পূর্ণ ওয়েবপেজের মূল কন্টেইনার।
  3️⃣ <head> → মেটা তথ্য, টাইটেল, স্টাইলশিট, স্ক্রিপ্ট থাকে।
  4️⃣ <body> → মূল কন্টেন্ট, টেক্সট, ইমেজ, লিংক, ভিডিও ইত্যাদি থাকে।
  5️⃣ <header> → শিরোনাম, লোগো, নেভিগেশন মেনু থাকে।
  6️⃣ <main> → ওয়েবসাইটের প্রধান কন্টেন্ট রাখার স্থান।
  7️⃣ <section> → আলাদা আলাদা অংশ বানানোর জন্য ব্যবহৃত হয়।
  8️⃣ <footer> → কপিরাইট, সোশ্যাল লিংক, অতিরিক্ত তথ্য রাখা হয়।
